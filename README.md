# Playwright Automation Framework (OrangeHRM)

## Overview

This is a professional-grade Playwright automation framework built using TypeScript.
It demonstrates real-world QA automation skills including:

* Page Object Model (POM)
* Authentication with storage state
* CRUD operations (Create, Read, Update, Delete)
* Data-driven testing
* HTML reporting
* Screenshot, video, and trace capture
* CI-ready structure

## Tech Stack

* Playwright
* TypeScript
* Node.js
* Dotenv

## Installation

```bash
npm install
npx playwright install
```

## Running Tests

Run all tests:

```bash
npm test
```

Run headed mode:

```bash
npm run test:headed
```

Run in Chrome only:

```bash
npm run test:chrome
```

Open report:

```bash
npm run report
```

Debug mode:

```bash
npm run debug
```

## Folder Structure

* tests/ → test cases
* pages/ → page objects
* utils/ → helpers
* playwright.config.ts → framework config
